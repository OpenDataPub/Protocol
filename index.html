Wouldn't it be great if all Open Data Platforms could share their data in one network?


Open for all - organized in trust

OpenDataPub is a decentralized protocol for Open Data Platforms to create an interchanging network. OpenDataPub lets every instance participate, but organises itself in a ring of trust approach.

Subscribe to everything your government does

OpenDataPub leverages its decentralized network to inform a user of any changes he's subscribed to. Tags, geolocation, document type and many more can instantly trigger a notification.

Empowered and compliant but not limited

Since OpenDataPub is a protocol it only set the minimum requirement. Every instance can add additional features to set itself apart. The baseline for metadata is the DCAT-AP Protocol by the european union.


Welcome to OpenDataPub! OpenDataPub is decentralized open data protocol based on protocols such as ActivityPub and ActivityStreams 2.0. OpenDataPub is in the stage of being a W3C Editors Draft. The protocol specifies the data structures and APIs for server and client to create a decentralized open data network, where instances can share the data the provide.
